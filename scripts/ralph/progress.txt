## Codebase Patterns
- **Styling**: Use NativeWind with Tailwind classes - `className="flex-1 bg-white"`
- **Conditional styles**: Use `cn()` from `src/lib/cn.ts` for merging classNames
- **NO className on**: CameraView, LinearGradient, Animated components - use inline `style` prop
- **Haptics**: Import from `expo-haptics`, use `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light)`
- **Animations**: Use `react-native-reanimated` v4 - FadeIn, FadeInDown for enter animations
- **Gestures**: Use `react-native-gesture-handler` - GestureDetector with Gesture API
- **State**: React Query for server state `useQuery({ queryKey, queryFn })`, Zustand for local
- **Icons**: Use `lucide-react-native` - e.g., `import { Home } from 'lucide-react-native'`
- **Typecheck**: Run `bunx tsc --noEmit` to verify
- **Pressable over TouchableOpacity**: Always prefer Pressable
- **Custom modals**: Don't use Alert.alert() - build custom modal components
- **Safe area**: Import from `react-native-safe-area-context`, not react-native

## Key Files
- `src/app/` - Expo Router file-based routes
- `src/app/_layout.tsx` - Root layout (never delete RootLayoutNav)
- `src/components/` - Reusable UI components
- `src/lib/cn.ts` - className merge utility
- `CLAUDE.md` - Project conventions and rules
